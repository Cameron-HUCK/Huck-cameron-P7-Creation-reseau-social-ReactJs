# Groupomania #

Groupomania est un réseau social pour les employés de la société Groupomania

# Pour installer / lancer le frontend # 
- npm install
- http://localhost:3000/

# Pour installer / lancer le backend #
- npm install
- node server / nodemon server
- http://localhost:4000

# Dossier et fichier fournis #
- .env 
- images
